In app-v3.html, build a Net Worth screen (FT-34).

1. DATASERVICE — add sync methods:
   getNetWorthItems(): return JSON.parse(localStorage.getItem('financeApp_networth') || '[]')
   saveNetWorthItems(items): localStorage.setItem('financeApp_networth', JSON.stringify(items))
   addNetWorthItem({ name, value, currency, category, type, notes }):
   // type: 'asset' or 'liability'
   // asset categories: Cash | Investments | Property | Vehicle | Other
   // liability categories: Mortgage | Loan | Credit Card | Other
   const item = { id: Date.now(), name, value: parseFloat(value),
     currency, category, type, notes: notes||'',
     updatedAt: new Date().toISOString() };
   push, saveNetWorthItems()
   // SPEC RULE 2: auto-snapshot on EVERY add/edit/delete
   this.autoSaveNetWorthSnapshot();
   return item
   updateNetWorthItem(id, updates):
   find, merge updates, updatedAt = new Date().toISOString(), saveNetWorthItems()
   this.autoSaveNetWorthSnapshot(); // auto-snapshot
   deleteNetWorthItem(id): splice, saveNetWorthItems()
   this.autoSaveNetWorthSnapshot(); // auto-snapshot
   autoSaveNetWorthSnapshot():
   // compute current net worth in default currency, then:
   const history = JSON.parse(localStorage.getItem('financeApp_netWorthHistory') || '[]');
   history.push({ date: new Date().toISOString().split('T')[0],
     netWorth: this.computeNetWorth() });
   const trimmed = history.slice(-24); // keep last 24 snapshots
   localStorage.setItem('financeApp_netWorthHistory', JSON.stringify(trimmed))
   computeNetWorth(): sum all asset values minus liability values,
   converted to defaultCurrency via CurrencyService.fallbackRates
   getNetWorthHistory(): return JSON.parse(localStorage.getItem('financeApp_netWorthHistory') || '[]')

   - Empty state per section: "No assets added yet." / "No liabilities added yet."

   DOES NOT AFFECT Financial Summary totals.
   Net Worth is a standalone snapshot — not linked to expenses/income.
