In app-v3.html, build a Savings Goals feature.

1. DATASERVICE — add these sync methods (localStorage only, no await needed):
   getGoals(): return JSON.parse(localStorage.getItem('financeApp_goals') || '[]')
   saveGoals(goals): localStorage.setItem('financeApp_goals', JSON.stringify(goals))
   addGoal({ name, targetAmount, currency, deadline, notes, emoji, currentSaved }):
   - Validate name >= 2 chars, unique case-insensitive
   - Validate targetAmount > 0
   - If deadline set, must be in the future
   const goal = { id: Date.now(), name, targetAmount: parseFloat(targetAmount),
     currency, deadline: deadline||"", notes: notes||"", emoji: emoji||"",
     currentSaved: parseFloat(currentSaved||0), createdAt: new Date().toISOString() };
   push, saveGoals(), return goal
   updateGoal(id, updates): find, merge, saveGoals()
   deleteGoal(id): splice, saveGoals()
   addToGoal(id, amount): find goal, goal.currentSaved += parseFloat(amount), saveGoals()

2. NAVIGATION — add nav item:
   { key: 'goals', icon: 'fas fa-piggy-bank', label: 'Goals' }

   - Deadline shown if set; "X days remaining" or "OVERDUE" if past deadline
